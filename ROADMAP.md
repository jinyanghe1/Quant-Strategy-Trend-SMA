# 项目 Roadmap

## 项目概述
基于Backtrader框架实现的简单移动平均线（SMA）趋势跟踪策略，支持多标的批量回测，自动生成交易日志和可视化图表。

---

## V1.0 - 核心功能（当前版本）

### ✅ 完成的功能
- [x] SMA 趋势跟踪策略核心实现
- [x] 多标的批量回测支持
- [x] 交易日志自动生成
- [x] 可视化图表生成

## 开发规范

### PR 审查清单
1. **功能对齐检查**
   - PR 实现的功能是否在 Roadmap 中明确列出
   - 是否有实现了 Roadmap 之外的功能（超出范围）

2. **质量检查**
   - 代码是否有完整的注释和文档
   - 是否包含相关的单元测试
   - 是否遵循项目编码规范

3. **完整性检查**
   - 功能是否完整实现（非半成品）
   - 是否更新了相关文档
   - 是否处理了边界情况

### 代码审计维度
- ✅ 功能完整性（对标 Roadmap）
- ✅ 代码质量（可读性、性能、安全性）
- ✅ 文档更新（README、注释、示例）
- ✅ 测试覆盖（单元测试、集成测试）
